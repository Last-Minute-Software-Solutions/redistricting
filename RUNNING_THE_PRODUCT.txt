# How To Run and Contribute To This Java Program


##Running The Program (Initial Setup):

###For Linux Users:

Download and install Java. Follow the tutorial here:
https://opensource.com/article/19/11/install-java-linux 
Download and install Git. Follow the tutorial here: https://git-scm.com/book/en/v2/Getting-Started-Installing-Git/ 
Download and install ant. Follow the tutorial here: 
https://ant.apache.org/manual/install.html 


###For Windows and Mac Users:

There is a good guide for getting started, and connecting to a repository here:
https://phoenixnap.com/kb/how-to-install-git-windows
Follow the guide to get started working on Git
Download and install ant. Follow the tutorial here: 
https://ant.apache.org/manual/install.html
